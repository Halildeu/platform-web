# Getting Started

## Installation

Install the core design system package:

```bash
pnpm add @mfe/design-system
```

For X Suite packages, install individually:

```bash
pnpm add @mfe/x-data-grid @mfe/x-charts @mfe/x-editor
```

## First Component

Wrap your application with `ThemeProvider` and start using components:

```tsx
import { ThemeProvider, Button, Input } from '@mfe/design-system';

function App() {
  return (
    <ThemeProvider>
      <Input placeholder="Email" />
      <Button variant="contained">Submit</Button>
    </ThemeProvider>
  );
}
```

## Theming

Customize the theme by passing `defaultAxes` to `ThemeProvider`:

```tsx
import { ThemeProvider } from '@mfe/design-system';
import type { ThemeAxes } from '@mfe/design-system';

const axes: Partial<ThemeAxes> = {
  appearance: 'light',
  density: 'comfortable',
  radius: 'md',
};

function App() {
  return (
    <ThemeProvider defaultAxes={axes}>
      {/* Your app */}
    </ThemeProvider>
  );
}
```

## TypeScript Support

All components ship with full TypeScript declarations.
Import the `ThemeAxes` type for type-safe theme configuration:

```ts
import type { ThemeAxes, ThemeAppearance, ThemeDensity } from '@mfe/design-system';
```
