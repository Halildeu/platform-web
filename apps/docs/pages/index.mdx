# @mfe/design-system

Enterprise-grade React component library with 100+ components,
6 X Suite packages, and a composable block system.

## Quick Start

```bash
pnpm add @mfe/design-system
```

```tsx
import { Button, ThemeProvider } from '@mfe/design-system';

export default function App() {
  return (
    <ThemeProvider>
      <Button variant="contained" color="primary">
        Get Started
      </Button>
    </ThemeProvider>
  );
}
```

## Architecture

The design system follows a layered architecture:

- **Core** — Theme tokens, utility hooks, base primitives
- **Components** — 112 production-ready UI components
- **X Suite** — Six domain-specific enterprise packages
- **Blocks** — Composable page-level building blocks

All packages share a single theme provider and are tree-shakeable.
