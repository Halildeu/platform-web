# X-Scheduler

Calendar and scheduling package with view components for
event management, resource booking, and timeline views.

## Quick Start

```tsx
import { Scheduler } from '@mfe/x-scheduler';

<Scheduler
  events={events}
  view="week"
  onEventCreate={handleCreate}
  onEventMove={handleMove}
/>
```

## Components

| Component | Purpose |
|-----------|---------|
| `Scheduler` | Full calendar with view switching |
| `DayView` | Single day hour grid |
| `WeekView` | 7-day columnar layout |
| `MonthView` | Monthly calendar grid |
| `AgendaView` | List-based event display |
| `ResourceView` | Resource allocation view |
| `SchedulerEventCard` | Draggable event block |
| `SchedulerToolbar` | Navigation and view controls |
| `EventForm` | Event create/edit form |
| `CurrentTimeIndicator` | Current time line overlay |
| `MiniMonthNav` | Compact month navigation sidebar |

## Views

Switch between views with the `view` prop:

```tsx
<Scheduler
  events={events}
  view="month"        // 'day' | 'week' | 'month' | 'agenda'
  resources={rooms}   // Required for resource view
/>
```

## Resource Scheduling

Assign events to resources (rooms, people, equipment):

```tsx
<Scheduler
  view="resource"
  resources={[
    { id: 'room-a', title: 'Conference Room A' },
    { id: 'room-b', title: 'Conference Room B' },
  ]}
  events={bookings}
/>
```

## Hooks

| Hook | Purpose |
|------|---------|
| `useScheduler` | Core scheduler state and navigation |
| `useRecurrence` | Recurrence rule expansion |
| `useConflictDetection` | Overlapping event detection |
| `useTimezone` | Timezone conversion utilities |
| `useEventDrag` | Drag-to-reschedule event handling |
| `useExternalDrop` | External item drop onto calendar |
| `useSchedulerVariants` | View variant configuration |
| `useResponsiveScheduler` | Mobile-responsive scheduler layout |

## Recurring Events

Use the `useRecurrence` hook for repeating events:

```tsx
import { useRecurrence } from '@mfe/x-scheduler';

const { expandedEvents } = useRecurrence(events);
```
