# Dashboard Reference App

Standalone KPI dashboard demonstrating @mfe/design-system + @mfe/x-charts integration.

## Components Used

- **Card, Text, Badge** — layout and typography
- **BarChart, LineChart, PieChart** — data visualization
- **ChartContainer** — loading/error/empty states
- **Stack, Divider** — composition primitives

## Quick Start

```bash
npx @mfe/create-app my-dashboard --template dashboard
cd my-dashboard
pnpm dev
```

## First-Value Time Target

Under 10 minutes from `create-app` to a working dashboard with mock data.

## Features Demonstrated

- KPI metric cards with trend indicators
- Interactive chart panels with filters
- Activity feed with real-time updates
- Responsive grid layout
- Theme switching (light/dark)
