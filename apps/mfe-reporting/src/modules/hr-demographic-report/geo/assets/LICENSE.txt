tr-provinces.geo.json — Türkiye 81 il (province) GeoJSON polygon dataset
=========================================================================

See README.md (in this directory) for attribution markup snippet.

Data summary:
- 81 features (full TR il coverage, plate codes TR-01..TR-81)
- Geometry type: MultiPolygon
- properties.name: Turkish display name (TitleCase, e.g. "İstanbul", "Adana")
- Coordinate system: WGS84 (EPSG:4326)

Notes:
- properties.code (TR-XX) is stamped at load time by `normalizeTRGeoJson.ts`
  using the alias dictionary in `../tr-provinces.ts`. The raw file does NOT
  carry ISO 3166-2 codes.
- Polygons are simplified by the upstream author for web render performance;
  precision is appropriate for choropleth at country scale, not for
  cadastral / boundary work.
