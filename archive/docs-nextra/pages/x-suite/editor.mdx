# X-Editor

Rich text editing package with Tiptap-based components for content
authoring, slash commands, mentions, and collaborative editing patterns.

## Quick Start

```tsx
import { RichTextEditor } from '@mfe/x-editor';

<RichTextEditor
  value={content}
  onChange={setContent}
  toolbar={['bold', 'italic', 'heading', 'link', 'image', 'table']}
/>
```

## Components

| Component | Purpose |
|-----------|---------|
| `RichTextEditor` | Full-featured WYSIWYG editor |
| `TiptapEditor` | Direct Tiptap integration editor |
| `EditorToolbar` | Configurable toolbar strip |
| `EditorMenuBubble` | Floating selection menu |
| `SlashCommandMenu` | Slash-command block insertion menu |
| `MentionList` | @mention autocomplete dropdown |
| `EditorTableMenu` | Table manipulation controls |
| `EditorLinkDialog` | URL insertion with preview |
| `EditorImageUpload` | Drag-and-drop image insertion |

## Toolbar Customization

Build custom toolbars by composing toolbar items:

```tsx
<RichTextEditor
  toolbar={[
    'bold', 'italic', 'underline',
    '|',
    'heading', 'bulletList', 'orderedList',
    '|',
    'link', 'image', 'table',
  ]}
/>
```

## Hooks

| Hook | Purpose |
|------|---------|
| `useEditor` | Core editor state and commands |
| `useTiptapEditor` | Tiptap-specific editor binding |
| `useSlashCommands` | Slash command registration and handling |
| `useMentions` | Mention autocomplete logic |
| `useEditorFormField` | Cross-package: editor as a form field |

## Tiptap Integration

For advanced use cases, access Tiptap directly:

```tsx
import { createTiptapEditorCore, isTiptapAvailable } from '@mfe/x-editor';

if (isTiptapAvailable()) {
  const core = createTiptapEditorCore({ extensions: [...] });
}
```

## Output Formats

The editor supports multiple serialization formats:

```tsx
<RichTextEditor
  outputFormat="html"    // 'html' | 'markdown' | 'json'
  value={content}
  onChange={setContent}
/>
```
