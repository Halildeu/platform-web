Primitive bileşenler, design system'in temel yapı taşlarıdır. Tüm enterprise ve pattern bileşenleri bu primitifler üzerine inşa edilmiştir.

## Button

```tsx
import { Button } from '@mfe/design-system';

<Button variant="primary">Kaydet</Button>
<Button variant="secondary">İptal</Button>
<Button variant="ghost">Detay</Button>
<Button variant="danger">Sil</Button>
<Button variant="primary" size="sm">Küçük</Button>
<Button variant="primary" size="lg">Büyük</Button>
<Button variant="primary" loading>Yükleniyor...</Button>
<Button variant="primary" disabled>Devre Dışı</Button>
<Button variant="primary" leftIcon={<PlusIcon />}>Ekle</Button>
```

## Modal & Dialog

```tsx
import { Modal } from '@mfe/design-system';

<Modal
  open={isOpen}
  onClose={() => setIsOpen(false)}
  title="Onay"
  size="md"
>
  <p>Bu işlemi onaylıyor musunuz?</p>
  <Modal.Footer>
    <Button variant="secondary" onClick={() => setIsOpen(false)}>İptal</Button>
    <Button variant="primary" onClick={handleConfirm}>Onayla</Button>
  </Modal.Footer>
</Modal>
```

## Toast & Notification

```tsx
import { useToast } from '@mfe/design-system';

function SaveButton() {
  const toast = useToast();

  return (
    <Button onClick={() => {
      save();
      toast.success('Kayıt başarılı');
    }}>
      Kaydet
    </Button>
  );
}
```

## Diğer Primitifler

| Bileşen | Açıklama |
|---------|----------|
| `Divider` | Yatay/dikey ayırıcı çizgi |
| `Spinner` | Yükleme animasyonu |
| `Tag` | Etiket bileşeni |
| `Progress` | İlerleme çubuğu |
| `Pagination` | Sayfalama bileşeni |
| `EmptyState` | Boş durum gösterimi |
| `ErrorBoundary` | Hata yakalama sınırı |
| `Portal` | React portal wrapper |
| `VisuallyHidden` | Erişilebilirlik için gizli metin |
| `FocusTrap` | Fokus tuzağı (modal, dialog) |
