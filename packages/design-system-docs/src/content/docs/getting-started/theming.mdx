## Tema Eksenleri

| Eksen | Seçenekler |
|-------|-----------|
| Appearance | light, dark, high-contrast |
| Density | compact, comfortable, spacious |
| Radius | sharp, rounded |
| Elevation | flat, raised |

## Dark Mode

```tsx
import { useTheme } from '@mfe/design-system';

function ThemeToggle() {
  const { axes, setAppearance } = useTheme();
  return (
    <button onClick={() => setAppearance(axes.appearance === 'dark' ? 'light' : 'dark')}>
      {axes.appearance === 'dark' ? 'Light' : 'Dark'}
    </button>
  );
}
```

## Density Yapılandırması

Density ekseni bileşenlerin padding ve spacing değerlerini kontrol eder:

```tsx
import { DesignSystemProvider } from '@mfe/design-system';

function App() {
  return (
    <DesignSystemProvider
      theme={{
        density: 'compact', // compact | comfortable | spacious
      }}
    >
      <YourApp />
    </DesignSystemProvider>
  );
}
```

| Density | Padding | Row Height | Font Size |
|---------|---------|-----------|-----------|
| compact | 4px 8px | 32px | 13px |
| comfortable | 8px 12px | 40px | 14px |
| spacious | 12px 16px | 48px | 15px |

## Radius Yapılandırması

```tsx
<DesignSystemProvider
  theme={{
    radius: 'rounded', // sharp | rounded
  }}
>
```

| Radius | Değerler |
|--------|---------|
| sharp | border-radius: 2px |
| rounded | border-radius: 8px |

## Elevation Yapılandırması

Elevation ekseni gölge ve kenar stillerini kontrol eder:

```tsx
<DesignSystemProvider
  theme={{
    elevation: 'raised', // flat | raised
  }}
>
```

## Programatik Tema Değişimi

```tsx
import { useTheme } from '@mfe/design-system';

function ThemePanel() {
  const { axes, setAxes } = useTheme();

  return (
    <div>
      <select
        value={axes.appearance}
        onChange={(e) => setAxes({ appearance: e.target.value })}
      >
        <option value="light">Light</option>
        <option value="dark">Dark</option>
        <option value="high-contrast">High Contrast</option>
      </select>

      <select
        value={axes.density}
        onChange={(e) => setAxes({ density: e.target.value })}
      >
        <option value="compact">Compact</option>
        <option value="comfortable">Comfortable</option>
        <option value="spacious">Spacious</option>
      </select>
    </div>
  );
}
```
