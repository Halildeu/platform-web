## Genel Bakış

Bu bölüm, design system'deki tüm bileşenlerin, hook'ların ve yardımcı fonksiyonların API referansını içerir. Her bileşen için prop tipleri, varsayılan değerler ve kullanım örnekleri sunulur.

## Import Yolları

Tüm bileşenler ve hook'lar ana paket üzerinden import edilir:

```tsx
// Bileşenler
import { Button, Input, Select, DataGrid } from '@mfe/design-system';

// Hook'lar
import { useTheme, useToast, usePermission, useFormContext } from '@mfe/design-system';

// Tipler
import type { ButtonProps, InputProps, DataGridColumn } from '@mfe/design-system';

// Token'lar
import '@mfe/design-system/tokens/build/tokens.css';
import '@mfe/design-system/tokens/build/dark-mode.css';
```

## Tip Tanımları

Tüm bileşen prop tipleri `Props` son ekiyle export edilir:

```tsx
import type {
  ButtonProps,
  InputProps,
  SelectProps,
  DataGridProps,
  DataGridColumn,
  ThemeAxes,
  Permission,
  Role,
} from '@mfe/design-system';
```

### Ortak Tipler

```tsx
// Tema eksenleri
interface ThemeAxes {
  appearance: 'light' | 'dark' | 'high-contrast' | 'system';
  density: 'compact' | 'comfortable' | 'spacious';
  radius: 'sharp' | 'rounded';
  elevation: 'flat' | 'raised';
}

// Boyut
type Size = 'xs' | 'sm' | 'md' | 'lg' | 'xl';

// Durum
type Status = 'info' | 'success' | 'warning' | 'danger' | 'neutral';

// İzin
type Permission = `${string}:${string}`;
```

## TypeDoc ile Oluşturulan Dokümantasyon

Detaylı API referansı için TypeDoc ile oluşturulan otomatik dokümantasyonu kullanabilirsiniz:

```bash
# TypeDoc çıktısını oluşturmak için
cd packages/design-system
npx typedoc --out docs/api src/index.ts
```
