## Import

```tsx
import { Input } from '@mfe/design-system';
```

## Kullanim

```tsx
<Input label="Ad Soyad" placeholder="Adinizi girin" />
<Input label="E-posta" type="email" required />
<Input label="Telefon" mask="+90 (___) ___ __ __" />
```

## Variants

```tsx
<Input variant="outlined" label="Outlined" />
<Input variant="filled" label="Filled" />
<Input variant="underlined" label="Underlined" />
```

## Validation

```tsx
<Input
  label="E-posta"
  type="email"
  error="Gecerli bir e-posta adresi girin"
/>

<Input
  label="Kullanici Adi"
  helperText="En az 3 karakter"
  minLength={3}
/>
```

## Prefix / Suffix

```tsx
<Input label="Fiyat" prefix="₺" suffix=".00" type="number" />
<Input label="Web Sitesi" prefix="https://" />
<Input label="Arama" prefix={<IconSearch />} />
```

## Masking

Giris maskesi ile formatlenmis veri girisi:

```tsx

<Input label="IBAN" mask="TR__ ____ ____ ____ ____ ____ __" />
<Input label="Tarih" mask="__/__/____" />
```

## Erisilebilirlik

- **Label**: `<label>` elementi `htmlFor` ile otomatik baglanir
- **Error**: `aria-invalid="true"` ve `aria-describedby` ile hata mesaji iliskilendirilir
- **Required**: `aria-required="true"` otomatik eklenir
- **Helper text**: `aria-describedby` ile yardimci metin iliskilendirilir

<Aside type="caution">
  `label` prop'u her zaman belirtin. Gorsel etiket istemiyorsaniz `hideLabel` prop'unu kullanin — bu durumda etiket ekran okuyucular icin korunur.
</Aside>

## Ornekler

### Arama Kutusu

```tsx
<Input
  label="Ara"
  hideLabel
  placeholder="Bilesen ara..."
  prefix={<IconSearch />}
  suffix={query && <IconClear onClick={() => setQuery('')} />}
  value={query}
  onChange={(e) => setQuery(e.target.value)}
/>
```

### Sifre Alani

```tsx
<Input
  label="Sifre"
  type="password"
  showPasswordToggle
  helperText="En az 8 karakter, 1 buyuk harf, 1 rakam"
/>
```
