## Import

```tsx
import { DataGrid } from '@mfe/design-system';
```

## Kullanim

```tsx
const columns = [
  { field: 'id', headerName: 'ID', width: 70 },
  { field: 'name', headerName: 'Ad', flex: 1 },
  { field: 'email', headerName: 'E-posta', flex: 1 },
  { field: 'role', headerName: 'Rol', width: 120 },
];

const rows = [
  { id: 1, name: 'Ahmet Yilmaz', email: 'ahmet@ornek.com', role: 'Admin' },
  { id: 2, name: 'Ayse Demir', email: 'ayse@ornek.com', role: 'Editor' },
];

<DataGrid columns={columns} rows={rows} />
```

## Siralama

```tsx
<DataGrid
  columns={columns}
  rows={rows}
  sortable
  defaultSort={{ field: 'name', direction: 'asc' }}
/>
```

## Filtreleme

```tsx
<DataGrid
  columns={columns}
  rows={rows}
  filterable
  filterMode="client"  // veya "server"
/>
```

## Sayfalama

```tsx
<DataGrid
  columns={columns}
  rows={rows}
  pagination
  pageSize={25}
  pageSizeOptions={[10, 25, 50, 100]}
/>
```

## Satir Secimi

```tsx
<DataGrid
  columns={columns}
  rows={rows}
  selectable
  selectionMode="multiple"  // veya "single"
  onSelectionChange={(selectedIds) => console.log(selectedIds)}
/>
```

## Gruplama

```tsx
<DataGrid
  columns={columns}
  rows={rows}
  groupable
  defaultGroupBy={['role']}
/>
```

## Export

```tsx
<DataGrid
  columns={columns}
  rows={rows}
  exportable
  exportFormats={['csv', 'xlsx', 'pdf']}
/>
```

## Erisilebilirlik

- **Keyboard Navigation**: Ok tuslari ile hucre navigasyonu
- **ARIA**: `role="grid"`, `role="row"`, `role="gridcell"` otomatik
- **Focus Management**: Hucre fokus yonetimi ve tab navigasyonu
- **Screen Reader**: Kolon basliklari ve siralama durumu duyurulur
- **Selection**: `aria-selected` ile secim durumu belirtilir

<Aside type="note">
  Buyuk veri setleri (10,000+ satir) icin sanal kaydirma (virtualization) otomatik olarak aktif olur.
</Aside>

## Server-Side Islemler

Buyuk veri setleri icin sunucu tarafli siralama, filtreleme ve sayfalama:

```tsx
<DataGrid
  columns={columns}
  rows={data}
  loading={isLoading}
  pagination
  paginationMode="server"
  rowCount={totalRows}
  page={page}
  pageSize={pageSize}
  onPageChange={setPage}
  sortingMode="server"
  onSortModelChange={handleSort}
  filterMode="server"
  onFilterModelChange={handleFilter}
/>
```
