## Naming Convention

Tüm bileşenler PascalCase ile adlandırılır ve `@mfe/design-system` paketinden named export olarak sunulur:

```tsx
import { Button, Input, DataGrid } from '@mfe/design-system';
```

## Durum Göstergeleri

Her bileşen dokümantasyon sayfasında durum göstergesi bulunur:

| Durum | Açıklama |
|-------|---------|
| **Stable** | Üretim ortamında kullanıma hazır |
| **Beta** | API değişebilir, geri bildirim bekleniyor |
| **Alpha** | Erken aşama, deneysel |
| **Deprecated** | Kullanımdan kaldırılacak, alternatif belirtilmiştir |
