## Hızlı Başlangıç

<Steps>

1. **Paketi yükleyin**

   <Tabs>
     <TabItem label="pnpm">
     ```bash
     pnpm add @mfe/design-system
     ```
     </TabItem>
     <TabItem label="npm">
     ```bash
     npm install @mfe/design-system
     ```
     </TabItem>
     <TabItem label="yarn">
     ```bash
     yarn add @mfe/design-system
     ```
     </TabItem>
   </Tabs>

2. **Provider'ı ekleyin**

   Uygulamanızın kök bileşenini `DesignSystemProvider` ile sarın:

   ```tsx
   import { DesignSystemProvider } from '@mfe/design-system';

   function App() {
     return (
       <DesignSystemProvider>
         <YourApp />
       </DesignSystemProvider>
     );
   }
   ```

3. **İlk bileşeninizi kullanın**

   ```tsx
   import { Button } from '@mfe/design-system';

   function MyPage() {
     return (
       <Button variant="primary" size="md">
         Merhaba Dünya
       </Button>
     );
   }
   ```

4. **Dark mode ekleyin (opsiyonel)**

   ```tsx
   import { DesignSystemProvider } from '@mfe/design-system';

   function App() {
     return (
       <DesignSystemProvider theme={{ mode: 'dark' }}>
         <YourApp />
       </DesignSystemProvider>
     );
   }
   ```

</Steps>
