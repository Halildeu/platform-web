## Build Health

The full build pipeline:

```
typecheck (tsc --noEmit)  ->  test (vitest run)  ->  build (tsup/vite)
         |                         |                      |
    EXIT 0 required          EXIT 0 required         EXIT 0 required
```

| Check | Tool | Gate |
|---|---|---|
| Type safety | `tsc --noEmit` | PR merge block |
| Unit + integration | Vitest | PR merge block |
| Build output | tsup / Vite | PR merge block |
| Lint | ESLint + Prettier | PR merge block |
| Commit format | commitlint | PR merge block |

## Visual Regression

| Tool | Scope | Trigger |
|---|---|---|
| **Chromatic** | Storybook stories (design-system, ui-kit) | Every PR |
| **Playwright screenshots** | Full-page snapshots (docs app, shell app) | Nightly + PR |

- Chromatic catches component-level visual diffs with 0.01% threshold
- Playwright screenshots catch layout regressions across pages
- Both require manual approval for intentional changes

## Security

| Tool | What it checks | Cadence |
|---|---|---|
| **Trivy** | Container image vulnerabilities | Every build |
| **Semgrep** | Static code patterns (XSS, injection, secrets) | Every PR |
| **ZAP** | OWASP Top 10 dynamic scan | Weekly + pre-release |
| **SBOM** | Software bill of materials generation | Every release |
| **npm audit** | Dependency vulnerabilities | Daily (Dependabot) |
| **Socket.dev** | Supply chain risk (typosquat, install scripts) | Every PR |

### Security Targets

- 0 critical / 0 high vulnerabilities in production dependencies
- SBOM published with every release for enterprise compliance
- Security advisories published within 48 hours of disclosure
