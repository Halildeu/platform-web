## Scale

Based on a 4px grid:

| Token | Value | Usage |
|-------|-------|-------|
| `spacing.0` | 0px | No spacing |
| `spacing.1` | 4px | Tight |
| `spacing.2` | 8px | Compact |
| `spacing.3` | 12px | Default gap |
| `spacing.4` | 16px | Standard |
| `spacing.6` | 24px | Comfortable |
| `spacing.8` | 32px | Spacious |
| `spacing.12` | 48px | Section |
| `spacing.16` | 64px | Page |

## Density Modes

Spacing adapts to the active density mode:

| Mode | Control Height (md) | Padding (md) | Gap (md) |
|------|-------|---------|-----|
| `compact` | 28px | 8px | 6px |
| `comfortable` | 32px | 12px | 8px |
| `spacious` | 40px | 16px | 12px |
