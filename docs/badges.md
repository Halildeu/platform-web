## Usage Notes

- Replace `{owner}/{repo}` with the actual GitHub organization and repository name.
- Update the coverage percentage to reflect the actual measured value from CI.
- The test badge can be made dynamic by integrating with a badge service that reads CI results.
- For dynamic badges based on CI output, consider using [shields.io endpoint badges](https://shields.io/endpoint) with a JSON endpoint generated by the CI pipeline.
